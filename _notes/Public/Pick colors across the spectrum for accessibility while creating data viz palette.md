While mono-tone has it's aesthetic appeal (and it's easier to create a palette), we need to think of color blindness and accessibility. 

While choosing colors, specially for data visualizations like stack area graph and pie chart, try picking colors across spectrum - of different hues, tints and tone. This helps in color differentiation, thus improving Accessibility.
